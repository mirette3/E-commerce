# amit_and_orange_e-commerce_app

#### Overview
- this project is an application on Amit/orange's 2 weeks flutter course.
- it's an E-commerce Application using Flutter for browsing and buying items.
###### Features
- login signup
- browse products
- see product details
- count items
#### Package using
- Dart
- State management(Bloc)
- Dio
- Conditional builder
